# 落地层规则

> 功能：将分析转化为具体行动，确定优先级
> 
> **阶段四强制调用**

---

## 调用规则

```
┌─────────────────────────────────────────────────────────────┐
│ 落地层强制调用（阶段四必须执行）                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ Step 1: 二八法则（必调）                                     │
│         识别哪些20%的动作能产生80%的效果                     │
│         文件：models/pareto.md                              │
│                                                             │
│ Step 2: 艾森豪威尔矩阵（必调）                               │
│         将动作按重要/紧急分类，确定执行顺序                   │
│         文件：models/eisenhower.md                          │
│                                                             │
│ 输出：高价值一击 + 强制断舍离                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

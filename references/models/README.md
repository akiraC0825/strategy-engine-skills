## 📋 问题类型与模型选择指南

### 快速选择表

| 问题类型 | 推荐模型组合 |
|---------|-------------|
| 🔥 危机排查 | 第一性原理 → 5Why → 矛盾分析 → 艾森豪威尔 |
| 💡 创新设计 | 第一性原理 → 正反合 → 六顶思考帽 → 二八法则 |
| ⚖️ 复杂决策 | 多元思维 → 六顶思考帽 → 反身性 → 艾森豪威尔 |
| 📈 效能提升 | MECE → 第二层思维 → 误判心理学 → 二八法则 |

### 信息不足时的补充问题

当输入信息不足以选择模型时，询问用户：

```
1. 【问题性质】您面临的是：
   □ 已有问题需要解决（危机排查）
   □ 需要创造新方案（创新设计）
   □ 需要在多个选项中选择（复杂决策）
   □ 需要优化现有流程（效能提升）

2. 【紧迫程度】这个问题：
   □ 非常紧急（需要立即行动）
   □ 有一定时间（可以系统分析）
   □ 长期规划（可以深入思考）

3. 【资源约束】您拥有的资源：
   □ 资金充足
   □ 时间充足
   □ 人力充足
   □ 以上都有限

4. 【风险偏好】您对风险的接受程度：
   □ 激进（愿意承担高风险）
   □ 稳健（平衡风险与收益）
   □ 保守（优先保证安全）
```

## 🔗 模型组合关系

### 经典组合

```
根因挖掘组合：
MECE → 5Why → 第一性原理
（MECE确保完整，5Why找到根因，第一性原理找到解决方案）

方案创新组合：
第一性原理 → 正反合 → 六顶思考帽
（第一性原理提出新方案，正反合处理反对意见，六顶思考帽完善细节）

风险控制组合：
反身性 → 黑天鹅 → 可证伪
（反身性识别泡沫，黑天鹅预判极端，可证伪设定止损）

执行转化组合：
二八法则 → 艾森豪威尔
（二八法则识别关键，艾森豪威尔确定顺序）
```

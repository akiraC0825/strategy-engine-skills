# 六顶思考帽

> 构思层模型 | 用于阶段二：深度拆解与推演

---

## 核心定义

用6种不同颜色的帽子代表6种不同的思维模式，引导人们从多角度、多层面探讨问题。这是"平行思维"的核心工具，由英国学者爱德华·德博诺博士于1985年提出。

**核心理念**：强调"能够成为什么"，而非"本身是什么"，是寻求一条向前发展的路，而不是争论谁对谁错。

## 代表人物

| 人物 | 背景 | 核心贡献 |
|------|------|---------|
| **爱德华·德博诺** | 英国心理学家、剑桥大学思维基金会主席 | 1985年提出六顶思考帽，开创"平行思维"方法。被公认为"创新思维之父"，影响了人类进程的250位杰出人物之一 |

### 核心著作

- **《六顶思考帽》**（Six Thinking Hats）：全球畅销的思维训练书籍，被译为37种语言，行销54个国家和地区
- **《水平思考法》**（Lateral Thinking）：全球创新思维理论的奠基之作，六顶思考帽是对本书的具体运用
- **《平行思考》**：深入探讨创新思考方式和思维工具

### 德博诺思维体系

德博诺博士构建了完整的创新思维方法论体系：

| 方法 | 定义 | 与六顶思考帽的关系 |
|------|------|-------------------|
| **水平思考** | 摆脱非此即彼思维方式的思考方法，致力于提出不同的看法 | 六顶思考帽是水平思考的具体操作工具 |
| **平行思维** | 将垂直思考（逻辑）和水平思考（创意）结合的思维方式 | 六顶思考帽是平行思维的实践框架 |
| **六顶思考帽** | 一种思维训练模式和全面思考问题的模型 | 核心应用工具 |

> "水平思考"一词已被收入《牛津英语大辞典》，成为创新思维的同义词。

## 核心概念

### 六顶帽子的含义

| 颜色 | 代表 | 思维模式 | 核心问题 | 关键词 |
|------|------|---------|---------|--------|
| ⚪ 白色 | 事实 | 客观、中立 | 我们已知什么？需要什么信息？ | 数据、信息、客观 |
| 🔴 红色 | 情感 | 直觉、情绪 | 我的感觉是什么？直觉告诉我什么？ | 感受、直觉、预感 |
| ⚫ 黑色 | 谨慎 | 批判、风险 | 有什么问题？会出什么错？ | 风险、困难、缺陷 |
| 🟡 黄色 | 价值 | 乐观、收益 | 有什么好处？为什么可行？ | 优点、价值、机会 |
| 🟢 绿色 | 创造 | 创新、可能性 | 有什么替代方案？如何改进？ | 创意、替代、可能 |
| 🔵 蓝色 | 控制 | 组织、总结 | 我们的议程是什么？下一步做什么？ | 控制、组织、决策 |

### 每顶帽子的详细解析

#### ⚪ 白色思考帽：信息与数据之帽

**特点**：
- 客观：只关注客观的事实和数据
- 中立：不带有任何主观判断和情感色彩

**作用**：
- 提供信息：收集和呈现与问题相关的所有事实、数据、信息
- 明确问题：帮助确定问题的现状和边界

**核心问题**：
1. 我们现在有什么信息？
2. 我们还需要什么信息？
3. 我们怎么得到所需要的信息？

**使用要点**：
- 只陈述事实，不带观点
- 区分"已知事实"和"需要了解的事实"
- 用数据和证据说话

**禁忌**：不要解释、不要推断、不要加入个人判断

#### 🔴 红色思考帽：情感与直觉之帽

**特点**：
- 情感：代表情绪、感觉和直觉
- 主观：基于个人的情感反应和主观印象

**作用**：
- 表达情感：让人们能够自由地表达自己对问题的感受和情绪
- 提供直觉：直觉有时可以为决策提供重要线索

**核心问题**：
1. 你对此的感觉是什么？
2. 你的直觉告诉你什么？
3. 你有什么预感？

**使用要点**：
- 表达直觉，无需理由
- 可以表达兴奋、担忧、期待、压力等任何感受
- 无需解释"为什么"

**禁忌**：不要要求"为什么"、不要质疑他人的感受

#### ⚫ 黑色思考帽：批判与风险之帽

**特点**：
- 谨慎：以批判和质疑的态度看待问题
- 风险：关注潜在的风险、问题和缺点

**作用**：
- 发现问题：找出方案中的漏洞和不足之处
- 评估风险：评估决策可能带来的风险和负面影响

**核心问题**：
1. 可能带来不好的结果是什么？
2. 有没有什么缺点？
3. 执行过程中会有什么困难？

**使用要点**：
- 逻辑批判，有理有据
- 关注真正的风险，而非消极抱怨
- 指出具体问题，而非笼统否定

**禁忌**：不要只是否定，要说原因；不要过度使用，避免陷入消极

#### 🟡 黄色思考帽：价值与收益之帽

**特点**：
- 积极：以乐观和积极的态度看待问题
- 优点：关注方案的优点、好处和价值

**作用**：
- 发现价值：找出方案中的积极因素和潜在价值
- 提供信心：增强对方案的信心和支持

**核心问题**：
1. 有哪些积极有利的条件？
2. 存在哪些有价值的方面？
3. 有没有什么特别吸引人的地方？

**使用要点**：
- 寻找价值，积极乐观
- 基于现实的乐观，而非空想
- 发现被忽视的好处

**禁忌**：不要空想，要基于现实；不要忽视风险（那是黑帽的工作）

#### 🟢 绿色思考帽：创造与可能之帽

**特点**：
- 创新：鼓励创造性思维和新想法
- 可能性：关注各种可能性和替代方案

**作用**：
- 产生创意：激发新的想法和解决方案
- 拓展思路：打破常规思维，开拓新的思考方向

**核心问题**：
1. 还有其他选择吗？
2. 新的想法、建议和假设是什么？
3. 其他可能的选择是什么？

**使用要点**：
- 激发创意，不评判对错
- 头脑风暴，求异思维
- 先发散再收敛

**禁忌**：不要批判，先发散再收敛；不要急于评估可行性

#### 🔵 蓝色思考帽：控制与组织之帽

**特点**：
- 控制：负责整个思考过程的组织和控制
- 管理：管理思考的流程和方向

**作用**：
- 组织思考：确定思考的顺序和重点
- 总结归纳：在思考结束时进行总结和归纳

**核心问题**：
1. 我们的目标是什么？
2. 下一步怎么办？
3. 我们要做出什么决定？

**使用要点**：
- 组织流程，控制节奏
- 定义问题和目标
- 总结关键发现和行动决定

**禁忌**：不要跳过，要有序进行；不要代替其他帽子的思考

### 平行思维 vs 对抗思维

```
┌─────────────────────────────────────────────────────────┐
│ 传统思维（对抗式）：                                     │
│ A说"应该这样做"，B说"不对，应该那样做"                   │
│ 结果：争论、对立、低效                                   │
│                                                         │
│ 平行思维（六顶思考帽）：                                 │
│ 所有人同时戴上同一顶帽子，朝同一个方向思考               │
│ 结果：协作、完整、高效                                   │
└─────────────────────────────────────────────────────────┘
```

**平行思维的核心优势**：
1. **避免对抗**：所有人朝同一方向思考，不再对立
2. **全面视角**：每个角度都被充分探索
3. **节省时间**：减少无意义的争论
4. **激发创意**：安全的环境中自由表达

## 核心方法论

### 使用顺序

#### 单次分析推荐顺序
```
蓝帽（定义问题）→ 白帽（收集事实）→ 红帽（表达感受）→ 
黄帽（发现价值）→ 黑帽（识别风险）→ 绿帽（创造方案）→ 蓝帽（总结决策）
```

#### 紧急决策简化顺序
```
蓝帽 → 白帽 → 黑帽 → 黄帽 → 蓝帽
```

#### 创新探索顺序
```
蓝帽 → 白帽 → 绿帽 → 黄帽 → 黑帽 → 绿帽 → 蓝帽
```

#### 项目规划顺序
```
蓝帽 → 白帽 → 绿帽 → 黄帽 → 黑帽 → 红帽 → 蓝帽
```

#### 风险评估顺序
```
蓝帽 → 白帽 → 黑帽 → 黄帽 → 绿帽 → 蓝帽
```

### 每顶帽子的使用要点

| 帽子 | 要点 | 禁忌 | 时间建议 |
|------|------|------|---------|
| 白帽 | 只陈述事实，不带观点 | 不要解释、不要推断 | 3-5分钟 |
| 红帽 | 表达直觉，无需理由 | 不要要求"为什么" | 1-2分钟 |
| 黑帽 | 逻辑批判，有理有据 | 不要只是否定，要说原因 | 3-5分钟 |
| 黄帽 | 寻找价值，积极乐观 | 不要空想，要基于现实 | 3-5分钟 |
| 绿帽 | 激发创意，不评判对错 | 不要批判，先发散再收敛 | 5-10分钟 |
| 蓝帽 | 组织流程，控制节奏 | 不要跳过，要有序进行 | 开头2分钟+结尾2分钟 |

### 团队使用规则

1. **一次只能戴一种颜色的帽子**
2. **必须"演什么像什么"** - 仅仅只从这种颜色帽子代表的特质来思考，决不掺杂其它颜色的特质
3. **所有人同步戴同一顶帽子** - 确保平行思维的效果
4. **由蓝帽主持控制节奏** - 确保有序进行

## 经典案例

### 案例1：ABM公司应对疯牛病危机

```
背景：1996年，欧洲最大牛肉生产公司ABM因疯牛病一夜之间丧失80%收入

六顶思考帽分析（12人，60分钟）：

🔵 蓝帽：目标是在没有收入的情况下生存6周

⚪ 白帽：成本结构分析、库存情况、员工数量、现金流数据

🔴 红帽：恐慌、焦虑，但也是变革的机会

🟡 黄帽：发现：
• 降低成本的方法30个
• 营销创意35个

⚫ 黑帽：筛选掉无用的后，剩下25个可行创意

🟢 绿帽：如何在危机中创新？转向其他肉类产品？

🔵 蓝帽：选择最可行的创意，立即执行

结果：靠着这25个创意，ABM度过6星期没有收入的日子
```

### 案例6：房地产公司建造决策

```
背景：基于经济状况良好、空置办公空间锐减的情况，某房地产公司董事们考虑是否建造新办公大楼

🔵 蓝帽：定义决策问题 - 是否建造新办公大楼

⚪ 白帽：
- 闲置办公空间趋势调查分析显示，闲置办公空间数量呈锐减趋势
- 董事会预计，待新办公大楼建成时，仍存在办公空间严重紧缺
- 政府预测显示，至少在大楼建设期间，经济仍将稳定增长

🔴 红帽：
- 一些董事认为初步拟建的大楼看起来非常丑陋
- 虽然方案极具成本效益，但担心人们会不喜欢在里面工作

⚫ 黑帽：
- 担心政府预测可能错误
- 市场经济可能进入"周期性衰退"期
- 如果办公建筑缺乏吸引力，在同等租金下，企业会选择更美观的大楼

🟡 黄帽：
- 如果经济发展保持平稳、政府预测正确，公司将取得很好收益
- 可能在衰退期到来之前卖掉房产
- 或与租户签订足以度过任何衰退期的长期租约

🟢 绿帽：
- 讨论是否需要改变建筑设计
- 让人们有更愉悦的体验

🔵 蓝帽：综合决策
```

### 案例7：医院护理质量管理

```
背景：护士长参加"六顶思考帽"培训后，在全科推广

案例：非计划拔管问题分析

⚪ 白帽（信息帽）：
- 什么时间、什么时段发生的
- 哪一层级的护士值班
- 患者的意识状态
- 是否有意识拔管、是否约束
- 有无陪人陪护
- 拔管后是否再置管
- 非计划拔管风险评分等客观数据

🔴 红帽（情感帽）：
- 患者及家属的不良情绪而不配合治疗？
- 护士的二次固定是否未落实？
- 打喷嚏、咳嗽等用力过猛导致？
- 护理人员健康宣教是否落实？

⚫ 黑帽（警示帽）：
- 现存留置胃管及尿管过程中的风险
- 护理过程中存在的困难
- 给予大家的警示

🟢 绿帽（创新帽）：
- 发挥护士的主观能动性
- 大胆运用创新思维
- 改变以往的固定方式
- 使二次固定既舒适又牢固
```

### 模型组合使用示例

#### 六顶思考帽 + 思维导图
```
1. 用思维导图作为记录工具
2. 每顶帽子创建一个分支
3. 在分支下展开相关想法
4. 最后整合成完整思维导图
```

#### 六顶思考帽 + SWOT分析
```
1. 先做SWOT分析（优势、劣势、机会、威胁）
2. 黄帽深入分析优势和机会
3. 黑帽深入分析劣势和威胁
4. 绿帽基于SWOT结果提出创新方案
5. 红帽表达对战略方向的直觉
6. 蓝帽做出最终决策
```

#### 六顶思考帽 + 鱼骨图
```
1. 黑帽阶段：用鱼骨图分析问题根因
2. 绿帽阶段：针对每个根因提出解决方案
3. 黄帽阶段：评估解决方案的价值
4. 黑帽阶段：评估解决方案的风险
5. 蓝帽阶段：选择最优方案
```

#### 六顶思考帽 + 头脑风暴
```
1. 白帽：明确头脑风暴的问题边界
2. 绿帽：进行头脑风暴（不批判）
3. 黄帽：筛选有价值的想法
4. 黑帽：评估可行性风险
5. 红帽：直觉判断哪个想法最有潜力
6. 蓝帽：确定下一步行动
```

## 德博诺思维框架模拟

### 德博诺的核心思想

```
德博诺思维体系 = 水平思考 + 平行思考 + 六顶思考帽

核心理念：
1. 思考是可以学习的技能
2. 创造性不是天赋，是可以训练的能力
3. 摆脱"非此即彼"的二元思维
4. 关注"能够成为什么"而非"本身是什么"
5. 用工具替代意志力，让系统思维成为习惯
```

### 水平思考的四个原则

1. **识别主导观念**：找出控制我们思考的隐性假设
2. **寻找替代方案**：不满足于唯一的"正确"答案
3. **打破模式**：用随机刺激、逆向思维等方法打破思维定式
4. **保持焦点**：始终围绕问题核心，不偏离目标

### 德博诺思维的日常训练

## 常见问题与解答

### Q1：六顶思考帽是否每次都必须全部使用？

**答**：不一定。可以根据实际情况调整：
- 简单决策：蓝→白→黑→黄→蓝
- 创新探索：蓝→白→绿→黄→黑→绿→蓝
- 风险评估：蓝→白→黑→黄→蓝

### Q2：红帽的直觉是否可靠？

**答**：红帽的价值在于：
- 捕捉被理性分析忽略的信息
- 表达潜在的情感和价值观
- 作为决策的重要参考（而非唯一依据）
- 直觉往往是经验的快速提取

### Q3：黑帽思维是否过于消极？

**答**：黑帽的价值在于：
- 不是消极，而是谨慎
- 提前发现问题，避免失败
- 有理有据的批判，而非情绪化否定
- 与黄帽平衡使用

### Q4：如何在争论激烈的会议中引入六顶思考帽？

**答**：
1. 先获得领导者支持
2. 从小问题开始实践
3. 强调"大家一起戴同一顶帽子"
4. 用规则代替意志力

### Q5：六顶思考帽能否用于个人思考？

**答**：完全可以：
- 个人决策同样需要多角度思考
- 可以画六个格子，分别填写
- 帮助避免思维盲点

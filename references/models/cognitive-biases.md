# 人类误判心理学

> 防守层模型 | 用于识别决策中的认知偏见

---

## 核心定义

人类决策常受25种心理倾向影响，导致非理性判断。识别这些倾向是避免误判的第一步。

## 核心方法论

### 高优先级自查清单（决策必查）

```
在重大决策前，逐项检查：

□ 激励偏见
  → 是否有利益冲突？
  → 这个建议对我有利还是对他有利？
  → "如果你想要说服别人，要诉诸利益，而非诉诸理性"

□ 过度自信
  → 我是否高估了自己的判断能力？
  → 我是否认为"这次不一样"？
  → 历史上类似情况的结果是什么？

□ 厌恶损失
  → 我是否因已投入而不愿放弃？
  → 如果从零开始，我还会做这个决定吗？
  → 沉没成本是否影响我的判断？

□ 寻求认同
  → 我是否只看支持自己观点的证据？
  → 我是否主动寻找了反对意见？
  → 验证性偏见是否存在？

□ 权威影响
  → 我是否因专家/权威而放弃独立判断？
  → 这个建议背后的激励是什么？
  → 权威的专长是否适用于这个问题？

□ 从众心理
  → 我是否因为别人都在做而跟随？
  → 大多数人相信的就一定是对的吗？
  → 我有独立的判断依据吗？
```

### 中优先级自查清单

```
□ 喜爱/憎恨倾向
  → 是否因情感而影响判断？
  → 是否因喜欢而忽视缺点？
  → 是否因讨厌而忽视优点？

□ 回馈倾向
  → 是否因小恩小惠而妥协？
  → 免费的东西是否有代价？

□ 妒忌倾向
  → 是否因他人成功而冲动？
  → 别人的成功能否复制到自己身上？

□ 简单联想
  → 是否有刻板印象？
  → 是否用"标签"代替思考？

□ 避免怀疑
  → 是否为了消除不确定性而过早决策？
  → 是否在压力下匆忙决定？
```

## 经典案例

### 联邦快递激励案例

```
问题：夜班工人总是不能按时完成工作

错误激励：
• 按小时计薪
• 工人想多加班多赚钱
• 结果：效率低下

正确激励：
• 按班次计薪
• 完成工作就可以提前下班
• 结果：效率大幅提升

教训：
→ 激励机制设计至关重要
→ 人们会对激励做出反应，而非意图
```

### 施乐公司销售案例

```
问题：新机器卖得不如性能低下的旧机器

错误激励：
• 销售旧机器提成更高
• 销售员自然推销旧机器
• 结果：劣等机器卖得更好

教训：
→ 激励与目标必须一致
→ 不要期望销售员违背自身利益
```

### 投资决策自我审视

```
场景：考虑投资某热门公司股票

误判检查：

1. 寻求认同：
   你只看看多的分析报告？
   → 检查：是否主动寻找了看空的观点？

2. 过度自信：
   你认为"这次不一样"？
   → 检查：历史上类似情况的结果是什么？

3. 厌恶损失：
   你是否因已投入研究时间而不愿放弃？
   → 检查：如果从零开始，你还会投资吗？

4. 简单联想：
   因为CEO有名，所以公司一定好？
   → 检查：CEO名气和公司长期价值有因果关系吗？

5. 回馈倾向：
   公司IR对你很好，所以你要投资？
   → 检查：这是投资决策还是情感回报？

6. 妒忌倾向：
   因为别人赚了很多，所以你也要投？
   → 检查：别人的成功能复制到你身上吗？

结论：识别出[X]个可能的问题倾向
      需要重点关注：[...]
```

## 模拟芒格思维

当需要模拟芒格思维方式时，使用以下框架：

```
【芒格模式激活】

核心问题：
• "我这个决策是否受到某种偏见的影响？"
• "我是否在做自己想做的事，还是应该做的事？"
• "如果我是完全理性的，我会怎么决定？"

检查清单（芒格常用）：
□ 激励是否合理？
□ 是否有确认性偏见？
□ 是否过度自信？
□ 是否受权威影响？
□ 是否受群体影响？

达尔文式逆向思维（芒格推荐）：
• 达尔文经常寻找可以推翻自己结论的证据
• 特别是在自己深信不疑的时候
• 这种习惯帮助他避免了很多错误

芒格式结论：
→ "反过来想，总是反过来想。"
→ "告诉我会死在哪里，我就永远不去那里。"
→ [...]
```

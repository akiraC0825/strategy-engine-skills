## 快速输出模板（简化版）

适用于信息充分、时间紧迫的场景：

```
═══════════════════════════════════════════════════════════════
                    ⚡ 快速分析报告
═══════════════════════════════════════════════════════════════

【问题】[一句话描述]
【类型】[危机排查/创新设计/复杂决策/效能提升]

【根因】[根本原因]
【方案】[推荐方案]
【风险】[主要风险]

═══════════════════════════════════════════════════════════════

🔥 立即执行：
1. [核心动作1]（完成标准：[...]，时间：[...]）
2. [核心动作2]（完成标准：[...]，时间：[...]）

🗑 绝对不做：
• [断舍离事项] → 原因：[...]

⚠️ Kill Metric：
• 如果[指标]达到[阈值]，立即[止损行动]

═══════════════════════════════════════════════════════════════
```

## 信息补充请求模板

当信息不足时使用：

```
═══════════════════════════════════════════════════════════════
                    ❓ 信息补充请求
═══════════════════════════════════════════════════════════════

为了为您选择最合适的分析方法，请补充以下信息：

【必填】
1. 核心问题是什么？
   （请用一句话描述您面临的问题）

2. 您期望得到什么？
   □ 问题诊断 + 根因分析
   □ 创新方案 + 实施路径
   □ 决策建议 + 风险评估
   □ 行动清单 + 优先级排序

【可选】（帮助更精准分析）
3. 时间紧迫性：紧急 / 中等 / 不急
4. 风险偏好：激进 / 稳健 / 保守
5. 资源情况：充足 / 一般 / 紧张

═══════════════════════════════════════════════════════════════

提示：您可以直接回答上述问题，或者告诉我"用默认设置分析"，
我将使用合理的默认假设进行分析。

═══════════════════════════════════════════════════════════════
```

## 默认假设声明模板

使用默认假设时必须包含：

```
═══════════════════════════════════════════════════════════════
                    ℹ️ 分析假设说明
═══════════════════════════════════════════════════════════════

由于您未提供部分信息，本次分析使用以下默认假设：

| 假设项 | 默认值 | 说明 |
|--------|--------|------|
| 时间紧迫性 | 中等 | 假设有几天到几周的分析时间 |
| 风险偏好 | 稳健 | 平衡风险与收益 |
| 资源情况 | 一般 | 假设资源有限但可用 |

如这些假设不符合您的实际情况，请告知，我将重新分析。

═══════════════════════════════════════════════════════════════
```

## 输出格式选择规则

| 场景 | 使用模板 | 说明 |
|------|---------|------|
| 标准分析 | 完整输出模板 | 四层分析完整展示 |
| 时间紧迫 | 快速输出模板 | 精简版，核心信息 |
| 信息不足 | 信息补充请求模板 | 先收集必要信息 |
| 使用默认值 | 默认假设声明 + 完整模板 | 必须说明假设 |

---

## 模板调用方式

在主SKILL.md中通过以下方式调用：

```markdown
输出格式请参考：[output-template.md](references/output-template.md)

根据分析场景选择合适的模板：
- 标准分析 → 使用「完整输出模板」
- 时间紧迫 → 使用「快速输出模板」
- 信息不足 → 先使用「信息补充请求模板」
```

# 防守层规则

> 功能：风险控制，设定失败标准，预防极端事件
> 
> **阶段三强制调用，无论阶段一是否选中防守层**

---

## 调用规则

```
┌─────────────────────────────────────────────────────────────┐
│ 防守层强制调用（阶段三必须执行）                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ Step 1: 可证伪原则（必调）                                   │
│         设计Kill Metric——明确的失败指标                     │
│         文件：models/falsifiability.md                      │
│                                                             │
│ Step 2: 黑天鹅理论（必调）                                   │
│         预判极端事件，设计反脆弱结构                          │
│         文件：models/black-swan.md                          │
│                                                             │
│ Step 3: 人类误判心理学（必调）                               │
│         识别决策中可能存在的认知偏见                          │
│         文件：models/cognitive-biases.md                    │
│                                                             │
│ Step 4: 反身性理论（选调）                                   │
│         当问题涉及市场/竞争/群体认知时调用                    │
│         文件：models/reflexivity.md                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 反身性理论调用条件

| 场景 | 是否调用 | 理由 |
|------|---------|------|
| 金融投资决策 | ✅ 调用 | 市场认知会影响价格 |
| 竞争分析 | ✅ 调用 | 竞争对手的认知会影响行为 |
| 品牌策略 | ✅ 调用 | 消费者认知会影响购买 |
| 内部管理 | ❌ 不调用 | 较少涉及群体认知反馈 |
| 技术方案设计 | ❌ 不调用 | 纯技术问题 |
| 危机排查 | ⚠️ 视情况 | 如果涉及公众舆论则调用 |

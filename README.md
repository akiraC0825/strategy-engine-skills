# strategy-engine-skills
strategy-engine-skills，一个搭载了人类顶级商业、科学与哲学思维模型的“战略拆解与执行引擎”。
本skills引用人类历史上主要的思维模型，分为四类，
>  * 破局层： 亚里士多德第一性原理、毛泽东矛盾分析法、MECE 原则、5Why 分析  
>  * 构思层： 黑格尔正反合、六顶思考帽、芒格多元思维/跨学科类比、第二层思维  
>  * 防守层： 波普尔可证伪原则、索罗斯反身性、塔勒布黑天鹅、人类误判心理学  
>  * 落地层： 帕累托二八法则、艾森豪威尔矩阵
> 当输入一个问题时，把复杂问题剥丝抽茧，按以上四个阶段输出，并强制转化为防弹级别的可执行动作。
